# THE-BUTTON
A Macropad with one singular button. Used for an emergency button to close a tab when a parent is coming perhaps...
The Button is a simple macropad, with a simple, 3D printed frame, and a PCB that can be hand soldered.
<img width="463" height="514" alt="Screenshot 2026-05-02 at 7 42 19 AM" src="https://github.com/user-attachments/assets/9bdbb472-41a1-442c-8d17-fe5018fbf8f7" />
It features a big, red button that can be pressed to a custom function (cmd-w, cmd-tab, cmd-t). I made this as a my firs wireless project, and my first project with a programmable LED. The PCB itself is rather simple, featuring some capacitors, and on/off switch, the switch, led, and the microcontroller, the Seeed Xiao ESP32-C3. <img width="622" height="646" alt="Screenshot 2026-05-02 at 7 46 33 AM" src="https://github.com/user-attachments/assets/ad70fc2c-68e9-4f55-9105-7984fa5f76f7" /> The Seeed module was chosen because it has such a small and compact size.
